# Project Sprint

## 最新版
* [v5.0](v5/v5_0_ja)

v5では、Project Sprint を単なるプロジェクト推進手法としてではなく、不確実な状況の中で価値を探索しつづけるためのフレームワークとして再整理しました。

探索と収束、多様な個人と共通了解、価値の湧現といった観点から、Project Sprint の背景にある世界観・構造・態度を体系的に記述しています。
