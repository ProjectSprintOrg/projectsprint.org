# Key Terms

> [!NOTE]
> **本章の位置づけ：**
>
> 本章では、Project Sprint において用いられる主要な用語の定義を示す。<br>
> これらの用語は、日常的な意味とは異なるニュアンスで使用される場合があるため、本書を通じて一貫した理解で捉えるための基準として提示する。
>
> **制作側の目的：**
>
> Project Sprint で用いられる重要な概念の意味を定義し、用語の解釈の揺れを防ぐ。
>
> **読者に提供する価値：**
>
> Framework を読む際に重要となる概念の意味を正確に理解し、文書内で使われる言葉を共通の理解で捉えられるようになる。

- **チーム**：共通の目的をもってプロジェクト内の会議体に参加し、目的に向かって出力を行うメンバーの集まり。
- **プロジェクトチーム**：プロジェクトの目的に対する共通了解をもち、その達成に向けて自律的に協働するメンバーの集まり。チームの一種だが、例外的に、全メンバーが参加する会議体を持たない場合や、出力を行わないアドバイザー的な立ち位置のメンバーを持つ場合がある。
- **プロジェクト**：プロジェクトチームが、プロジェクト外部の社会に対して何らかの変化や価値を提供するために、自発的に行う活動。
- **活動**：チームやメンバーが、意志や目的をもって、ある状態を別の状態へと変化させることを目的として行う身体性を伴う行動であり、出力を必ず含む。
- **出力**：プロジェクトメンバーが行動して作成物を生み出す行為。
- **成果**：チームが、出力の結果としてプロジェクト内外にもたらした変化であり、価値の実現に向けた結果。
- **価値**：プロジェクトの成果を通じて、外部のステークホルダーにとって意味や便益として認識されるもの。
- **作成物**：チームメンバーによって生み出され、チームに共有されて他のメンバーが解釈や反応を行うことが可能な、具体的な形をもつもの。
